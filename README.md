# Student Management System

Description

A Python program to manage a basic student record system from the console. It allows adding, displaying, searching, and deleting students.

Features

Add one or multiple students
Display all registered students
Search for a student by ID
Delete a student by ID

Data Structure

Students are stored in a list. Each student is represented as a dictionary with the following fields:

id
name
age
course

Validations

Ensures numeric inputs are valid
Prevents empty inputs
Handles cases when the list is empty
